# LOL-match-discord-bot
- 리그 오브 레전드의 프로 경기에 대한 정보를 크롤링해주는 디스코드 봇입니다.

## 가상환경
```bash
# 가상환경 구축
$ python3 -m venv venv

# 가상환경 실행
$ cd venv
/venv$ source bin/activate
(venv) ~/venv$

# 가상환경 종료
(venv)$ cd venv/bin
(venv) ~/venv/bin$ deactivate
```

## 패키지
```bash
# 패키지 설치
$ pip3 install -r requirements.txt
# 패키지 확인
$ pip3 freeze
```

## 실행
```bash
$ python3 discordboy.py
```
